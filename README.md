## Introduccion
Pacman es un juego donde controlas un personaje que come capsulas.
El personaje principal tiene que escapar de 4 fantasmas.

Fantasmas:
Fantasma Rojo,
Fantasma Verde,
Fantasma Azul,
Fantasma Amarillo.
